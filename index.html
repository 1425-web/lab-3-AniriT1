Каскадные таблицы стилей CSS (Cascading Style Sheets) определяют представление документа, его внешний вид. Стиль в CSS представляет правило, которое указывает браузеру, как надо форматировать элемент. Форматирование может включать установку цветов фона элемента, настройку шрифта, обрамления и т. д

Иногда для одних и тех же элементов требуется различная стилизация. И в этом случае мы можем использовать классы.

Для идентификации уникальных на web-странице элементов используются идентификаторы, которые определяются с помощью атрибутов id.

Кроме селекторов тегов, классов и идентификаторов в CSS также есть так называемый универсальный селектор, который представляет знак звездочки (*). Он применяет стили ко всем элементам на HTML-странице.
